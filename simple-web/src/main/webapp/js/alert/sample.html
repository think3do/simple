jQuery alert Plugin

이 플러그인은 브라우저 시스템 알림 창을 조금 더 이쁘게게 화면에 표시하고 다양한 브라우저에서 동일하게 표시 하기 위한 플러그인입니다.

기본 jQuery가 설치되어 있어야 하며 3가지(Alert, Confirm, Prompt)의 기능을 제공합니다

Alert

$.alert('내용',{옵션})

Option

옵션명 설명 값
title 알림창의 타이틀 변경 html
em 강조 문구 html
callEvent 확인 후 이벤트 Function
btnName.confirm 확인 버튼 텍스트 변경 String

Example

$.alert('안녕하세요! 빌리블입니다<br>info@billible.co.kr로 궁금한 점은 문의하세요',{em:'Billible',title:'인사'})

Confirm

$.confirm('내용',{옵션})

Option

옵션명 설명 값
title 알림창의 타이틀 변경 html
em 강조 문구 html
callEvent 확인 후 이벤트 Function
cancelEvent 취소 후 이벤트 Function
confirmButton 확인 버튼 텍스트 변경 String
cancelButton 취소 버튼 텍스트 변경 String

Example

$.confirm('안녕하세요! 빌리블입니다<br>info@billible.co.kr로 문의하시겠습니까?',{
 em:'Billible',
 title:'안내',
 callEvent:function(){$.alert('감사합니다.')},
 cancelEvent:function(){$.alert('예 궁금한건 언제든지 문의 주세요');},
 cancelButton:'다음에'
})

Prompt

$.prompt('내용',{옵션})

Option

옵션명 설명 값
title 알림창의 타이틀 변경 html
em 강조 문구 html
callEvent 확인 후 이벤트 Function(val) val : 입력값
cancelEvent 취소 후 이벤트 Function
btnName.confirm 확인 버튼 텍스트 변경 String
btnName.cancel 취소 버튼 텍스트 변경 String
input input 타입 변경 버튼 텍스트 변경 text(기본),tel,numberpassword

Example

$.prompt('1234를 입력해주세요',{
 title:'인증',
 callEvent:function(val){
  if(val==1234){
   $.alert('비밀번호가 맞습니다');
  } else {
   $.alert('비밀번호를 정확히 입력해주세요');}
  },
 input:'password',
 confirmButton:'인증하기'
 })

체인 연결

jQuery 체인 이벤트 후 실행

Example

$("#item-box").stop().animate({"margin-left":"500px"},1000)
.animate({"margin-left":"0"},300)
.delay(300).alert('1번 더!!',
{
  callEvent:function(){
   $("#item-box").stop().animate(
    {"margin-left":"500px"},1000
   ).animate(
    {"margin-left":"0"},300
    )
  }
});
